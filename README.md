## Installation

If you want to run the scraper manually on your own machine:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/funnymonke0/simple-isef-scraper
   ```
2. ***Install Dependencies***
   ```bash
   cd simple-isef-scraper
   pip install -r requirements.txt
   ```
3. ***Run***
   ```bash
   python run.py
   ```
